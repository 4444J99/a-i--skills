# Labanotation Symbol Reference

## Staff Structure

```
The Labanotation staff represents the body:

     ┌───┬───┐
     │ L │ R │  ← Head (top)
     ├───┼───┤
     │ L │ R │  ← Arms
     ├───┴───┤
     │   │   │  ← Body center
     ├───┼───┤
     │ L │ R │  ← Legs
     └───┴───┘
        ↓
       Time flows upward
```

## Direction Symbols

### Basic Directions

```
Forward (place):     ▢ (rectangle)
Backward:            ◇ (diamond)
Right side:          ▷ (right triangle)
Left side:           ◁ (left triangle)
Place (neutral):     ○ (circle, or short rectangle)
```

### Diagonal Directions

```
Forward-Right:    ⬒ (rectangle with right diagonal)
Forward-Left:     ⬓ (rectangle with left diagonal)
Backward-Right:   ⬔ (diamond with right diagonal)
Backward-Left:    ⬕ (diamond with left diagonal)
```

## Level Indicators

Shading indicates level (height):

```
HIGH (above shoulder):     ■ (filled/black)
MIDDLE (waist to shoulder): ▤ (hatched/striped)
LOW (below waist):         □ (empty/white)
```

Combined with direction:
```
Forward High:    ▮ (filled rectangle)
Forward Middle:  ▦ (hatched rectangle)
Forward Low:     ▯ (empty rectangle)
```

## Duration

Symbol length = duration:

```
Short:   ─   (1 beat)
Medium:  ──  (2 beats)
Long:    ─── (4 beats)

Symbols are stretched vertically on the staff
```

## Body Parts

### Support Columns (Legs/Feet)

```
     │
  ┌──┴──┐
  │L│ │R│  ← Support columns (next to center line)
  │ │ │ │
```

### Leg Gesture Columns

```
     │
  ┌──┴──┐
│L│ │ │ │R│  ← Leg gestures (outside support)
```

### Arm Columns

```
     │
  ┌──┴──┐
│L│    │R│  ← Arm columns (outermost)
```

## Timing Marks

```
│  Bar line (measure start)
╫  Double bar (section end)
═  Accent
→  Hold/sustain
```

## Relationship Pins

```
⦿  Facing direction pin
⦾  Contact
⦶  Relationship to partner
```

## Movement Qualities

### Effort Qualities (appended to direction symbols)

```
/   Sudden (quick)
\   Sustained (slow)
~   Light
≈   Strong
```

### Pathway Symbols

```
─→  Straight path
⟿   Curved path
~~~  Indirect/meandering
```

## Common Movements

### Walk Forward

```
     ▮     ← Forward high (step)
  ▯     ← Forward low (supporting leg)
  │  │
```

### Jump

```
  ▯  ▯    ← Both feet leave ground
  │  │    ← Support column empty
```

### Turn

```
  ↻ or ↺   ← Rotation direction
  With rotation amount:
  90°, 180°, 360°
```

### Arm Gesture

```
In arm column:
  ▮  ← Right arm forward high
     │
```
