# Source Evaluation Rubrics

## CRAAP Test

| Criterion | Score 1-5 | Questions to Ask |
|-----------|-----------|------------------|
| **Currency** | | When was it published/updated? |
| | 5 | Published within last year, regularly updated |
| | 3 | Published within 5 years |
| | 1 | Over 10 years old, no updates |
| **Relevance** | | Does it relate to your research? |
| | 5 | Directly addresses research question |
| | 3 | Tangentially related |
| | 1 | Only loosely connected |
| **Authority** | | Who is the author/publisher? |
| | 5 | Recognized expert, peer-reviewed |
| | 3 | Professional with some credentials |
| | 1 | Unknown author, no credentials |
| **Accuracy** | | Is it supported by evidence? |
| | 5 | Well-cited, verifiable claims |
| | 3 | Some citations, mostly verifiable |
| | 1 | No citations, unverifiable claims |
| **Purpose** | | Why does this exist? |
| | 5 | Inform/educate, balanced view |
| | 3 | Mix of information and promotion |
| | 1 | Primarily to sell or persuade |

**Total Score Interpretation:**
- 20-25: High quality, use as primary source
- 14-19: Acceptable, verify key claims
- 8-13: Use cautiously, seek corroboration
- 5-7: Avoid as primary source

## Bias Detection Checklist

### Author Bias
- [ ] Check author's affiliation and funding
- [ ] Look for conflict of interest disclosures
- [ ] Review author's other publications
- [ ] Consider author's ideological position

### Publication Bias
- [ ] Is it peer-reviewed?
- [ ] What's the journal's reputation?
- [ ] Is it a predatory journal?
- [ ] Check retraction status

### Content Bias Signals
- [ ] Emotional or loaded language
- [ ] One-sided presentation
- [ ] Cherry-picked data
- [ ] Missing alternative viewpoints
- [ ] Unattributed claims
- [ ] Logical fallacies

### Red Flags
⚠️ No author listed
⚠️ No publication date
⚠️ Extraordinary claims without evidence
⚠️ URL doesn't match claimed organization
⚠️ No references or citations
⚠️ Grammatical errors in formal publications
